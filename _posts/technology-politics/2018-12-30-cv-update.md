## Relevant Projects

### Miyagi-Studio Framework
**Language/Technology: PHP & Wordpress**
Developing a multi-purpose theme for Wordpress, using Redux, a backend PHP framework, targeted at website designers to give more dashboard controls over the website look-and-feel by embedding more creative open-source technologies, like WebGL. Utilised a version-controlled source and build structure with gulp, Node.js and yarn for deployment and packaging on a local server, embedded within Wordpress. This was a self-driven project.

### Continuous Integration and Deployment Lead for VST3 Production Suite
**Language/Technology: Git, Gitlab CI**
In order to have versioned VST3 software working on multiple platforms and for various digital-audio-workstations, I engineered a Gitlab-CI pipeline focusing on hidden keys and anchors to reduce code redundancy.

### Deployment & Continuous Integration Engineering for VISR
**Language/Technology: Git, Gitlab CI**
To achieve versioned cross-platform releases of the [VISR](http://www.s3a-spatialaudio.org/visr) framework, an object-based audio rendering framework for the open-source community, I worked with the University of Southampton remotely to craft an automated pipeline with Gitlab-CI, from building in CMake, Unit testing through to packaging.

### Software Developer on Spatial Audio Software
**Language/Technology: Max/MSP, C++ and JUCE**
In order to produce software for object-based audio, we utilised an ecosystem with JUCE framework and our object-based audio rendering framework. I Engineered and integrated research that lead to the implementation of a VST software that allows audio ‘objects’ to be panned for multiple speaker setups within digital audio workstations. Tested with up to 42 speakers. Much of the code will be openly available on GitHub for open-source-community.

### Kluster, A Max/MSP framework for OpenGL
**Language/Technology: Max/MSP; GLSL**
Self-lead MSc project to create a number of abstracted extendable modules that give more user-friendly control over the way audiovisuals are rendered in real-time.
