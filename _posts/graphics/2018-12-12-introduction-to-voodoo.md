Voodoo is a small framework for that receives various forms messages and translates them into.

Like many frameworks out there, many of the functions contained in Voodoo are quite simply some useful encoding and deconding functions, as well as translations from different types.

The motivation behind Voodoo is that because everyone is obsessed with React at the moment, it is a good opportunity to see how it can be used with a more create framework, notably the super cool `three.js`, which utilises WebGL by Kronos.
